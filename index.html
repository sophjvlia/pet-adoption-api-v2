API Documentation

Register User

POST /signup

Register a new user with email and password.

Login User

POST /login

Login an existing user with email and password.

Get dog breeds

GET /dog/breeds

Retrieve dog breeds.

Get cat breeds

GET /cat/breeds

Retrieve all cat breeds.

Create pet

POST /pets

Create a new pet.

Get pets

GET /pets

Retrieve all pets.

Get a single pet

GET /pets/:id

Retrieve a pet based on ID.

Update pet

PUT /pets/:id

Update a pet based on ID.

Delete pet

DELETE /pets/:id

Delete a pet based on ID.
